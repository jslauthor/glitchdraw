# Welcome to Glitch Draw

This is a fun little side project I worked on for Webflow's office. 

Qt + Smart LED Matrix + Raspberry Pi + IR Touch Bezel = FUN

Parts:
* 6x Smart LED Matrix
* Rasperry Pi 4
* GPIO splitter board
* Voltage Amplifier
* 3D Printed Frame
* IR Touch Bezel

Software:
* Raspbian Buster + GBM Drivers
* Qt 5.12
